Bem vindo ao repositório github para acompanhamento das aulas git-code-education

Utilizamos o editor Markdown para gerar esta página e explicar como foram os procedimentos e comandos para o primeiro push do projeto, ou seja a partir de um repositório local em nossa máquina subimos o controle de versão para este repositório github.

$ mkdir git-code-education
$ cd git-code-education
$ touch README.md
$ git init
#editamos o arquivo
$ vim README.md
$ git add README.md
$ git commit -m "primeira alteracao de arquivo readme"
$ git remote add origin https://github.com/rcadecaro/git-code-education.git
$ git push origin master
